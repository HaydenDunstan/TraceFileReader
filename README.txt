# Trace File Reader
A .cap tracefile reader designed as a project for a university assignment in a third year Computer Networks course

The Program will:
1. Read the .cap file
2. Print the total number of connections
3. Print spicific details of each connection
4. Print a general summary of the TCP connections
5. Print data about the complete TCP connections

The program tracefile_reader.py will not run without packet_struct.py and connections_struct. A sample .cap file has been provided in the repository. Code will compile automatically.

**To run tracefile_reader.py the code input the following line and replace [.CAP] with your chosen .cap file:**

Windows:

py tracefile_reader.py [.CAP]

Linux and Mac:

python3 tracefile_reader.py [.CAP]
